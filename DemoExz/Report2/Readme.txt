деление на test и train:
Обоснование разбиения набора данных: разбиение набора данных на обучающую и тестирующую выборки будут произведенны на основании официального сайта функций машинного обучение - scikit-learn. Данная библиотека предоставляет метод разбиения данных https://scikit-learn.org/stable/modules/generated/sklearn.model_selection.train_test_split.html, в котором демонстрируется реккомендуемая выборка 1/3. Данное значение можно обусловить тем, что для обучения требуется больше данных, чем для тестирование.
